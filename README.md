# DolPort - Open Port Scanner

## Overview
DolPort is a simple, yet powerful port scanner that allows users to scan for open ports on a remote IP address. It's built using Python and is optimized for use on Linux and Termux terminals.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sohamdev77/DolPort.git
   cd DolPort
Install the dependencies:

pip install -r requirements.txt
Usage
Run the script:

python dolport.py
Enter the target IP address and port range to scan.

Example

Enter the IP address: 192.168.1.1
Enter the port range (e.g., 1-100): 20-50
